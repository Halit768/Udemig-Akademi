# Udemig-Akademi Web Sitesi
Bu proje, farklı ülkelerde eğitim veren Udemig-Akademi hakkında bilgi veren basit bir web sitesidir. HTML ve CSS kullanılarak oluşturulmuştur.

## Kullanılan Teknolojiler
- HTML
- CSS

## Özellikler
- Akademinin hizmet verdiği ülkeler ve sunulan eğitim programları hakkında bilgi.
- Akademi hakkında genel bilgiler ve iletişim bilgileri.
